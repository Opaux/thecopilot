A dedicated low latency telemetry/alert system for autonomous vehicles (cars, drones, ground rovers, maritime vessels) powered by a Zynq SoC and PetaLinux. To use, upload the BOOT.bin, image.ub, and boot.scr to an SD card and insert into a Zybo Rev. B. Hardware needed: 

- Zybo Rev B.
- SD Card
- Pmod NAV
- Pmod GPS
- Pmod OLED
- Ethernet Cable
